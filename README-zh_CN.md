# 瓜大助手

![NPUHelper Logo](./NPU_Helper_banner.svg)

瓜大的百科全书

[English Introduction](/README.md)

# 主要特征

+ 实时天气查询
+ 学校校车时间表查询 (支持智能节假日)
+ 校历和放假上班安排查询（还带一个校园新闻）
+ 学校的一些典型地点介绍
+ 全学校几乎所有单位部门的电话、办公地址查询
+ 图书馆服务
+ 一卡通余额和流水查询

# 支持

+ 右至左布局支持
+ 控件英文支持

# 提示

+ 对于第三方提供的内容，可能只支持简体中文
+ 本应用并非由西北工业大学开发或者维护，此应用的内容亦无法表示西北工业大学的观点或者态度。
